# 天玑 TianJi 1.0 使用说明

天玑是一款用于飞行器仿真 CSV 数据可视化的小型桌面软件。它支持多 CSV 导入、三维轨迹显示、默认状态量九宫格、自定义对比图窗、曲线样式管理、工程保存与 PNG 导出。

## 启动软件

在项目目录下运行：

```powershell
cd D:\szt\NKUpro\study\TianJiVision\code\flight_csv_visualizer
python main.py
```

启动后主界面左侧为三维轨迹视图，右侧为二维图窗选项卡，底部为公共图例。

## 导入 CSV

1. 点击菜单 `文件 -> 打开 CSV`。
2. 选择一个或多个 CSV 文件。
3. 在导入窗口中确认每个文件的别名。
4. 如果软件无法自动识别必要列，会弹出列映射窗口，可手动指定 `time/x/y/z/vx/vy/vz/roll/pitch/yaw/wx/wy/wz` 等标准变量对应的 CSV 列。

软件会记住上一次导入 CSV 的文件夹，下次打开时会默认进入该目录。

当前版本默认最多加载 10 个 CSV 文件，单个 CSV 最多 100000 行。

## 查看默认图窗

默认选项卡为 `基本状态量`，包含 3 x 3 图窗：

- X/Y/Z 方向速度
- 滚转角、俯仰角、偏航角
- X/Y/Z 轴角速度

每个图窗右下角有一个小三角操作入口，点击后可打开：

- `曲线`：管理当前图窗曲线
- `设置`：修改标题、坐标轴标签、网格、图例、线宽、线型和坐标轴范围
- `导出`：导出当前二维图窗 PNG

## 使用三维轨迹

左侧三维区域会根据 CSV 的 `x/y/z` 映射绘制轨迹。

点击三维区域上方的 `曲线管理` 可设置：

- 每个文件轨迹是否显示
- 颜色
- 线宽
- 线型
- 透明度
- 绘制层级顺序
- 是否启用各坐标轴等比例

启用“各坐标轴等比例”后，软件会保持三维显示中的单位尺度一致，同时坐标轴范围仍按数据上下界自动调整。

## 公共图例

底部公共图例显示所有已加载 CSV 文件。

- 勾选框：控制该文件全局显示/隐藏
- 颜色块：修改该文件默认颜色
- 双击别名：重命名文件别名

全局显示状态会影响三维轨迹和所有二维图窗。图窗内曲线管理中的局部显示状态会按最后一次操作优先生效。

## 新建自定义选项卡

点击二维选项卡右上角的 `+` 按钮可创建自定义选项卡。

创建时可设置：

- 选项卡名称
- 行数
- 列数

自定义选项卡支持 1 x 1 到 5 x 5 图窗布局。默认 `基本状态量` 选项卡不可关闭，自定义选项卡可以关闭。

## 配置自定义图窗

在自定义图窗右下角操作入口中点击 `设置`。

`绘图设置` 中可设置：

- 图窗标题
- X/Y 轴标签
- 网格
- 局部图例
- 默认线宽和线型
- 手动坐标轴范围
- 横向对比设置
- 纵向对比设置

X/Y 轴标签提供常用单位选项，例如位置 m、速度 m/s、温度 K、力 N、能量 J、加速度 m/s^2、角度 rad/deg、角速度 rad/s/deg/s，也可以手动输入。

## 横向对比

横向对比用于比较多个 CSV 文件中的同一类变量。

1. 在自定义图窗设置中点击 `横向对比设置`。
2. 选择 `Y轴目标变量`。
3. 在 `参与文件与列映射` 表格中确认每个 CSV 使用哪一列。
4. 没有匹配列的文件默认不参与，也可以手动指定列。
5. 点击 `确定` 后应用设置。

适合比较多个仿真案例的同一状态量，例如所有文件中的 `vx`、`z` 或 `pitch`。

## 纵向对比

纵向对比用于比较同一个 CSV 文件中的多个变量。

1. 在自定义图窗设置中点击 `纵向对比设置`。
2. 选择一个数据文件。
3. 选择 X 轴列。
4. 勾选多个 Y 轴列。
5. 点击 `确定` 后应用设置。

适合查看同一案例中多个状态量的变化关系。

## 手动添加曲线

在自定义图窗设置的 `添加曲线` 页中可手动添加曲线。

需要选择：

- 数据文件
- X 轴列
- Y 轴列
- 曲线名称
- 是否使用文件颜色
- 颜色
- 线宽
- 线型

添加后可在图窗的 `曲线` 管理窗口中继续调整。

## 曲线管理

二维图窗和三维轨迹都支持曲线管理。

可调整内容包括：

- 显示/隐藏
- 名称或别名
- 颜色
- 线宽
- 线型
- 透明度
- 上移/下移

表格中越靠上的曲线绘制优先级越高，会覆盖靠下的曲线。

## 保存和加载工程

使用菜单：

- `文件 -> 保存工程`
- `文件 -> 工程另存为`
- `文件 -> 加载工程`

工程文件为 JSON 格式，保存内容包括：

- CSV 文件路径
- 文件别名和颜色
- 列映射
- 图窗配置
- 曲线样式和显示状态
- 自定义选项卡

工程文件不保存 CSV 数据本身。如果加载工程时 CSV 路径失效，软件会提示重新定位或跳过。

## 导出图片

可通过菜单或图窗操作入口导出 PNG：

- `导出 -> 导出三维轨迹 PNG`
- `导出 -> 导出当前二维图窗 PNG`
- 单个二维图窗右下角操作入口中的 `导出`
